# Admin Dashboard
- A modern and responsive Admin Dashboard built with React and Material UI.

## Features
- Responsive Design
- Light / Dark Mode
- 11 Dashboard Pages
- Interactive Charts
- Data Tables using MUI DataGrid
- React Router Navigation
- Reusable Components
- Modern UI with Material UI

## Technologies Used
- React.js
- Material UI (MUI)
- React Router
- JavaScript (ES6+)
- Vite

## Live Demo
https://dashboard-admin-ten-iota.vercel.app

## Installation

```bash
npm install
npm run dev
```
